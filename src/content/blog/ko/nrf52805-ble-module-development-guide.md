# nRF52805 기반 초소형 BLE 모듈 개발: 설계부터 양산까지의 초격차 디테일

초소형 무선 모듈 개발은 단순히 회로를 연결하는 작업이 아니라, 물리적 한계와 제조 공정의 특성을 완벽히 이해해야 하는 과정입니다. 본 포스트는 nRF52805 모듈 개발의 전 과정을 엔지니어링 관점에서 심층 분석한 기록입니다.

---

## 1. 초기 전략: 칩셋 및 PCB 레이어 최적화

- **칩셋 선정**: 초소형 폼팩터를 위해 nRF52805 WL-CSP 패키지 채택.
- **레이어 전략**: 핀 수가 적어 **2Layer**로 설계를 시작했으나, 부품 실장 밀도와 GND 평면 확보를 위해 **4Layer** 전환 가능성을 상시 검토하며 진행.
- **부품 소형화**: 모든 수동 소자를 **0603(0201 inch)** 사이즈로 적용하여 모듈 면적을 최소화했습니다.

## 2. H/W 설계 차별화: 신뢰성 확보를 위한 고급 기법

단순한 기능 구현을 넘어, 장기적인 신뢰성을 위해 다음과 같은 고급 설계 기법을 적용했습니다.

- **HPL(Hole Plugged and Lacquered) 공법**: 
    - 모든 Via에 HPL을 적용하여 RF IC 및 주변 부품의 납땜 시 발생하는 가스(Gas) 배출을 차단했습니다. 이는 냉납(Cold Solder) 발생을 막아 RF 성능의 일관성을 보장합니다.

![Via HPL 및 패턴 설계](../../../assets/images/blog/nrf52805-via-hpl-rf-pattern-layout.webp)
*그림 3: 신뢰성 확보를 위한 Via HPL 처리 및 RF 패턴 레이아웃*

- **X-tal Noise Isolation**: 
    - 32MHz 크리스탈 주변의 Copper를 분리(Isolation)하여 주변 고주파 노이즈가 클럭 안정성에 미치는 영향을 최소화했습니다.
- **RF 경로 최적화**: 
    - IC 출력단에서 안테나까지의 패턴을 대각선 일자로 배치하여, 기존 굴곡진 패턴 대비 전송 손실을 획기적으로 개선했습니다.

## 3. 제조 공정 딥다이브: Half Hole 이슈와 해결

### **Half Hole(Castellated Hole) 설계 표준**
측면 납땜용 Half Hole 가공 시 도금 유실 문제를 해결하기 위해 제조사와의 긴밀한 협의를 거쳐 다음 표준을 정립했습니다.

| 구분 | 설계 사양 | 비고 |
| :--- | :--- | :--- |
| **드릴(Drill)** | 0.4mm | 라우팅 드릴(1.0mm) 간섭 방지 |
| **패드(Pad)** | 0.6mm (최소) | 에칭 시 도금 생존성 확보를 위한 최소 수치 |
| **보강 기법** | Pad on Via | 측면 손상 시에도 전기적 연결성 확보 |

### **에칭 및 두께 제어**
- **에칭 공법**: 일반적인 판넬 도금 방식보다 정밀한 제어가 가능한 공법을 검토하여 Half Hole의 형상 유지력을 높였습니다.
- **두께 관리**: 자재 자체는 0.8T를 사용하되, 도금 및 표면 처리를 포함한 최종 두께가 0.9T 내외가 될 것을 계산하여 설계에 반영했습니다.

## 4. 7차에 걸친 CAD 검토 히스토리

본 프로젝트는 완성도를 높이기 위해 총 7차례의 정밀 CAD 리뷰를 거쳤습니다.

| 차수 | 주요 검토 및 수정 내용 |
| :--- | :--- |
| **1~2차** | 쉴드캔(Shield CAN) 부품 기구 및 고정 패드(Non-plate) 배치 검토 |
| **3~4차** | RF 단 구간 최소화 및 안테나-GND 간격 이격(Isolation) 강화 |
| **5차** | Y1 크리스탈 90도 회전 배치 및 배선 최단 거리 최적화 |
| **6차** | 메인보드(EVB)와 모듈 패드 간의 실장 정밀도 정합 확인 |
| **7차** | 쉴드캔 납땜 마스크 영역 및 Silk 최종 오타 검수 |

## 5. 실측 데이터 기반 RF 튜닝

- **Antenna Matching**: Pi(π) 매칭 회로(L3, C7, L4)를 통해 임피던스 정합.
- **성능 개선**: 초기 웨이브(Wave) 형상 안테나가 GND와 너무 가까워 이득(Gain)이 저하되는 현상을 발견, 안테나 폭 축소 및 GND 이격 거리 확장을 통해 방사 효율을 개선했습니다.

## 6. 양산 검사 지그(JIG) 및 소프트웨어 로직

양산 품질을 위해 정밀 소켓 지그와 전용 테스트 로직을 개발했습니다.

### **쇼트/오픈 검사 알고리즘**
모든 GPIO의 연결 상태를 자동으로 확인하는 소프트웨어 시퀀스를 적용했습니다.
1. 모든 I/O를 Pull-up 상태로 설정.
2. 특정 I/O 하나를 Out-mode(Low 출력)로 설정.
3. 나머지 I/O를 Input-mode로 읽어 Low가 입력되는 핀이 있는지 확인 (I/O 간 쇼트 감지).
4. 모든 핀에 대해 순차적으로 반복 수행.

### **RF 품질 검사 (DTM)**
- **Direct Test Mode**: 특정 채널에서 Carrier를 출력하여 스펙트럼 아날라이저로 주파수 정확도 확인.
- **발열 방지**: 테스트 중 과전류 발생 시 즉시 전원을 차단하는 보호 회로를 테스트 보드에 통합했습니다.

## 결론: 엔지니어링은 디테일에 있다

nRF52805 모듈 개발의 성공은 0.1mm의 패드 오차를 잡아내고, 보이지 않는 Via 내부의 가스 배출까지 고려한 디테일에서 결정되었습니다. 본 기록이 소형 무선 장치를 설계하는 동료 엔지니어들에게 실질적인 가이드라인이 되기를 바랍니다.
