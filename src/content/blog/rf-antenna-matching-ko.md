## 임피던스 매칭이 중요한 이유

RF 시스템에서 안테나와 RF IC 사이의 임피던스가 일치하지 않으면 반사손실(Return Loss)이 발생하여 송수신 성능이 크게 저하됩니다. 특히 BLE 모듈의 경우 PCB 안테나를 사용할 때 이 문제가 두드러집니다.

대부분의 RF IC는 **50Ω 출력 임피던스**를 기준으로 설계되어 있습니다. 안테나가 50Ω에서 벗어날수록 S11(반사계수)이 나빠지고, 실질적인 방사 전력이 줄어듭니다.

## VNA로 S11 측정하기

Vector Network Analyzer(VNA)는 안테나 임피던스를 측정하는 핵심 장비입니다.

```
측정 절차:
1. VNA 포트 캘리브레이션 (SOL: Short-Open-Load)
2. 측정 주파수 범위 설정 (예: 2.3GHz ~ 2.5GHz)
3. 안테나 급전점에 SMA 프로브 연결
4. S11 스미스 차트 확인
5. 목표: S11 < -10dB @ 2.4GHz
```

## 매칭 회로 설계

일반적으로 L-network 또는 π-network를 사용합니다.

### L-network 설계 순서

1. VNA에서 안테나의 **언매칭 임피던스** 읽기 (예: 35 - j15Ω)
2. 스미스 차트에서 50Ω로의 매칭 경로 결정
3. 직렬 인덕터 + 병렬 커패시터 값 계산
4. 0402 사이즈 칩 부품 실장 후 재측정

### 실전 팁

- 매칭 부품 값을 ±10% 범위에서 조정하면서 반복 측정
- 주변 금속(케이스, 배터리)의 영향을 반드시 고려할 것
- 최종 매칭은 **완성된 기구물 상태**에서 진행

## Nordic nRF52 기반 BLE 모듈 사례

nRF52832 기반 커스텀 BLE 모듈에서의 실제 매칭 결과:

| 조건 | S11 (dBm) | 통신 거리 |
|------|-----------|---------|
| 매칭 전 | -4.2 dB | ~8m |
| L-network 적용 | -18.7 dB | ~35m |
| 최적 매칭 완료 | -24.3 dB | ~50m |
